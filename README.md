# ddevs

A web project using Vite, TailwindCSS, and various JavaScript libraries.

## Installation

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Build the project for production:

```bash
npm run build
```

## Preview

Preview the production build locally:

```bash
npm run preview
```

## Dependencies

- Vite
- TailwindCSS
- FontAwesome
- Leaflet
- SweetAlert2
- idb
- ky
